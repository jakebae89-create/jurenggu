# Play 스토어 등록 문구 — 주령구(酒令具) 주사위

## 앱 이름 (최대 30자)
주령구(酒令具) 주사위

## 짧은 설명 (최대 80자)
신라 14면체 주사위 주령구! 친구들과 굴려서 나온 벌칙을 수행하는 전통 술자리 놀이.

## 전체 설명 (최대 4000자)
천년 전 신라인들이 즐기던 놀이를 손안에서 즐겨보세요.

「주령구(酒令具) 주사위」는 경주 동궁과 월지(안압지)에서 출토된 신라시대 14면체 나무 주사위 ‘주령구’를 3D로 되살린 술자리·모임 놀이 앱입니다. 정사각형 6면과 정육각형 8면, 총 14개의 면에는 각각 재미있는 벌칙이 새겨져 있습니다.

■ 이렇게 즐겨요
1. 참가자 이름을 등록합니다 (최대 14명).
2. 차례가 된 사람이 ‘굴리기’ 버튼을 누릅니다.
3. 3D 주사위가 굴러가고, 멈춘 면의 벌칙을 수행합니다.
4. 다음 사람에게 차례가 넘어갑니다.

■ 주요 기능
• 실제 유물 비율을 살린 3D 14면체 주사위 — 손가락으로 돌리고 확대/축소할 수 있어요.
• 신라시대 원형 벌칙 14종 수록 (한자 + 한글 읽기 + 풀이 제공).
• 벌칙 내용을 자유롭게 수정 — 우리 모임만의 규칙으로 바꿔보세요.
• 참가자 등록과 자동 차례 진행.
• 인터넷 없이도 실행되는 가벼운 앱.
• 설정과 참가자는 기기에만 저장되어 안심.

■ 수록된 전통 벌칙 예시
• 금성작무(禁聲作舞) — 소리 없이 춤추기
• 음진대소(飮盡大笑) — 다 마시고 크게 웃기
• 공영시과(空詠詩過) — 시 한 수 읊기
• 자창괴래만(自唱怪來晚) — ‘괴래만’ 노래 부르기

친구, 동료, 가족 모임에서 어색함을 깨고 분위기를 띄우는 데 제격입니다. 신라인의 멋과 풍류를 오늘의 자리에서 즐겨보세요!

※ 본 앱은 전통 술자리 문화를 소재로 하며 음주 관련 표현이 포함됩니다. 음주는 성인이 본인 책임하에 적절히 즐기시기 바랍니다.
